Why Compare Languages?

Understanding algorithms at multiple levels of abstraction helps you:

  • Pseudocode: Focus on logic without syntax details
  • C: Understand memory management and pointer manipulation
  • Python: See the algorithm in its most readable form

The same algorithm, three perspectives‚Äîeach reveals different insights about data structures.

Height and Balance Factor

üìù Pseudocode

FUNCTION Height(node):
    IF node = NULL THEN
        RETURN -1
    END IF
    RETURN node.height
END FUNCTION

FUNCTION BalanceFactor(node):
    IF node = NULL THEN
        RETURN 0
    END IF
    RETURN Height(node.left) 
         - Height(node.right)
END FUNCTION

FUNCTION UpdateHeight(node):
    node.height ‚Üê 1 + MAX(
        Height(node.left),
        Height(node.right)
    )
END FUNCTION

‚öôÔ∏è C

void update_height(AVLNode *n) {

}

Right Rotation (LL Case)

üìù Pseudocode

FUNCTION RotateRight(y):
    // y is the unbalanced node
    x ‚Üê y.left
    B ‚Üê x.right
    
    // Perform rotation
    x.right ‚Üê y
    y.left ‚Üê B
    
    // Update heights
    // (child first!)
    UpdateHeight(y)
    UpdateHeight(x)
    
    // Return new root
    RETURN x
END FUNCTION

//     y              x
//    / \            / \
//   x   C    ‚Üí    A    y
//  / \                / \
// A   B              B   C

‚öôÔ∏è C

Rebalance Function

üìù Pseudocode

FUNCTION Rebalance(node):
    IF node = NULL THEN
        RETURN NULL
    END IF
    
    UpdateHeight(node)
    bf ‚Üê BalanceFactor(node)
    
    // Left-heavy
    IF bf > 1 THEN
        IF BalanceFactor(node.left) < 0
            node.left ‚Üê RotateLeft(
                node.left)
        END IF
        RETURN RotateRight(node)
    END IF
    
    // Right-heavy
    IF bf < -1 THEN
        IF BalanceFactor(node.right) > 0
            node.right ‚Üê RotateRight(
                node.right)
        END IF
        RETURN RotateLeft(node)
    END IF
    
    RETURN node
END FUNCTION

‚öôÔ∏è C

}

AVL Insertion

üìù Pseudocode

FUNCTION Insert(node, key):
    // Base case: create node
    IF node = NULL THEN
        RETURN CreateNode(key)
    END IF
    
    // Recurse left or right
    IF key < node.key THEN
        node.left ‚Üê Insert(
            node.left, key)
    ELSE IF key > node.key THEN
        node.right ‚Üê Insert(
            node.right, key)
    ELSE
        // Duplicate: no insert
        RETURN node
    END IF
    
    // Rebalance on way up
    RETURN Rebalance(node)
END FUNCTION

‚öôÔ∏è C

AVLNode *avl_insert(
    AVLNode *node, int key) {
    // Base case
    if (node == NULL) {
        return create_node(key);
    }
    
    // Recurse
    if (key < node->key) {
        node->left = avl_insert(
            node->left, key);
    } else if (key > node->key) {
        node->right = avl_insert(
            node->right, key);
    } else {
        // Duplicate
        return node;
    }
    
    // Rebalance
    return rebalance(node);
}

AVL Deletion

üìù Pseudocode

FUNCTION Delete(node, key):
    IF node = NULL THEN
        RETURN NULL
    END IF
    
    IF key < node.key THEN
        node.left ‚Üê Delete(
            node.left, key)
    ELSE IF key > node.key THEN
        node.right ‚Üê Delete(
            node.right, key)
    ELSE
        // Found node to delete
        IF node.left = NULL OR
           node.right = NULL THEN
            temp ‚Üê node.left
            IF temp = NULL THEN
                temp ‚Üê node.right
            END IF
            FREE node
            RETURN temp
        END IF
        // Two children
        succ ‚Üê FindMin(node.right)
        node.key ‚Üê succ.key
        node.right ‚Üê Delete(
            node.right, succ.key)
    END IF
    
    RETURN Rebalance(node)
END FUNCTION

The Real Insight

The algorithm is the same in all three languages. What changes is how we express memory allocation, pointer manipulation and type handling. Understanding all three makes you a better programmer.

Summary
